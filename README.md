# Echo news app

Your voice, amplified.

- [Live demo](https://echo-news-app.vercel.app)

## Setup

### Requirments

- Node.js >= 20
- Yarn >= 1.22.19

### Install dependencies

```bash
yarn
```

### Running the app

```bash
yarn dev
```

### Run linters

```bash
yarn lint
```

### Run linters and fix errors

```bash
yarn lint:fix
```
