# Echo news app

Your voice, amplified.

## Environments

- [Production](https://echo-news-app.vercel.app)
- [Development](https://echo-news-app-staging.vercel.app)

## Setup

### Requirments

- Node.js >= 20
- Yarn or npm

### Install dependencies

```bash
yarn
```

### Running the app

```bash
yarn dev
```

### Run linters

```bash
yarn lint
```

### Run linters and fix errors

```bash
yarn lint:fix
```
